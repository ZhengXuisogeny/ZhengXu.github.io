Introduction to Isogeny-Based Cryptography

What is Isogeny-Based Cryptography

Isogeny-based cryptography is a type of cryptographic technique that relies on elliptic curve isogenies and offers the advantage of short sizes.

SIDH

  1. Origin (2011)

    Luca De Feo and David Jao first proposed SIDH (Supersingular Isogeny Diffie-Hellman), a key exchange protocol based on supersingular elliptic curve isogenies. SIDH leaks image points, which may lead to security vulnerabilities.

    Towards quantum-resistant cryptosystems from supersingular elliptic curve isogenies

  2. Active Attacks on SIDH (2016)

    Steven D. Galbraith, Christophe Petit, Barak Shani, and Yan Bo Ti first proposed active attacks against SIDH.

    On the Security of Supersingular Isogeny Cryptosystems

  3. Passive Attacks on SIDH (2016, 2021)

    ：Christophe Petit proposed the Torsion Point Attack, which was later improved by Victoria de Quehen, Peter Kutas, Chris Leonardi, Chloe Martindale, Lorenz Panny, Christophe Petit, and Katherine E. Stange. However, these attacks require a significant parameter asymmetry between the communicating parties. (These attacks transform the isogeny computation problem into solving equations in quaternion algebras.)

    Faster Algorithms for Isogeny Problems Using Torsion Point Images Improved Torsion-Point Attacks on SIDH Variants

  4. Another Attack (2021)

    : Peter Kutas, Simon-Philipp Merz, Christophe Petit, and Charlotte Weitkamper proposed a new attack leveraging the ability of quantum computers to solve the Hidden Shift Problem. However, this attack did not break SIDH completely.

    One-Way Functions and Malleability Oracles: Hidden Shift Attacks on Isogeny-Based Protocols

  5. Breaking SIDH with Special Initial Curves (2023)

    : Wouter Castryck and Thomas Decru broke SIDH instances with special initial curves (j-invariant 1728) using Kani's theorem for higher-dimensional abelian varieties.

    An Efficient Key Recovery Attack on SIDH

  6. Analysis of General SIDH (2023)

    : Luciano Maino, Chloe Martindale, Lorenz Panny, Giacomo Pope, and Benjamin Wesolowski analyzed general SIDH instances and developed some attacks, though none are polynomial-time.

    A Direct Key Recovery Attack on SIDH

  7. Complete Break of SIDH (2023)

    : Damien Robert achieved a complete break of SIDH using 8-dimensional abelian varieties.

  8. At this point, SIDH, which was a fourth-round candidate in NIST's post-quantum cryptography standardization process, has been completely broken.

Modified Versions of SIDH

  1. M-SIDH (2023)

    ：Tako Boris Fouotsa, Tomoki Moriya, and Christophe Petit proposed M-SIDH, which hides image points to achieve security.

    M-SIDH and MD-SIDH: Countering SIDH Attacks by Masking Information

  2. FESTA Encryption (2023)

    ：Andrea Basso, Luciano Maino, and Giacomo Pope proposed FESTA, an encryption scheme that hides image points. Kohei Nakagawa and Hiroshi Onuki later proposed QFESTA, an optimized version.

    FESTA: Fast Encryption from Supersingular Torsion Attacks

    QFESTA: Efficient Algorithms and Parameters for FESTA Using Quaternion Algebras

  3. bin-SIDH (2023)

    Andrea Basso and Tako Boris Fouotsa proposed bin-SIDH and ter-SIDH, which achieve security by selecting different subgroups.

    New SIDH Countermeasures for a More Efficient Key Exchange

  4. These modified schemes require very large parameters to resist attacks on SIDH, making them impractical.

CSIDH

  1. Origin (2018)

    Wouter Castryck, Tanja Lange, Chloe Martindale, Lorenz Panny, and Joost Renes first proposed CSIDH (Commutative SIDH), a key exchange protocol based on supersingular elliptic curve isogenies over \\(\\mathbb{F}_p\\). CSIDH uses elliptic curves over finite fields \\(\\mathbb{F}_p\\), similar to ordinary curves with complex multiplication, and involves the action of ideal class groups (\\(\\mathbb{Z}[\\pi_p]\\) or \\(\\mathbb{Z}[\\frac{1+ \\pi_p}{2}]\\)), which are vulnerable to quantum attacks.

    CSIDH: An Efficient Post-Quantum Commutative Group Action

  2. Signature Schemes CSI-FiSh and SeaSign (2019)

    Ward Beullens, Thorsten Kleinjung, and Frederik Vercauteren proposed CSI-FiSh, a signature scheme based on CSIDH. Luca De Feo and Steven D. Galbraith proposed SeaSign. The main difference is that CSI-FiSh uses a cyclic ideal class group.

    CSI-FiSh: Efficient Isogeny Based Signatures Through Class Group Computations

    SeaSign: Compact Isogeny Signatures from Class Group Actions

  3. SiGamal Encryption (2020)

    Ward Beullens, Thorsten Kleinjung, and Frederik Vercauteren proposed CSI-FiSh, a signature scheme based on CSIDH. Luca De Feo and Steven D. Galbraith proposed SeaSign. The main difference is that CSI-FiSh uses a cyclic ideal class group.

    SiGamal: A Supersingular Isogeny-Based PKE and Its Application to a PRF

  4. Attack on the DDH Problem in CSIDH (2020)

    Wouter Castryck, Jana Sotakova, and Frederik Vercauteren attacked the DDH problem in CSIDH using quadratic characters (genus theory).

    Breaking the Decisional Diﬃe-Hellman Problem for Class Group Actions Using Genus Theory

  5. Attacking CSIDH Using SIDH Techniques (2023)

    Wouter Castryck, Marc Houben, Simon-Philipp Merz, Marzio Mula, Sam van Buuren, and Frederik Vercauteren used correspondences between groups generated by eigenvectors of the Frobenius endomorphism, computed torsion points by pairing, to attack CSIDH using SIDH techniques. However, due to limitations of the Tate pairing, this attack is not effective.

    Weak Instances of Class Group Action Based Cryptography via Self-pairings

  6. OSIDH (2020)

    Leonardo Colò and David Kohel proposed OSIDH, which differs from CSIDH in that it uses an order of an imaginary quadratic field instead of \\(\\mathbb{Z}[\\pi_p]\\) or \\(\\mathbb{Z}[\\frac{1+ \\pi_p}{2}]\\) for its endomorphism ring.

    Orienting Supersingular Isogeny Graphs

  7. Attack on the DDH Problem in OSIDH (2022)

    Wouter Castryck, Marc Houben, Frederik Vercauteren, and Benjamin Wesolowski attacked the DDH problem in OSIDH.

    On the Decisional Diffie–Hellman Problem for Class Group Actions on Oriented Elliptic Curves

  8. The hardness problems in CSIDH and OSIDH reduce to the Hidden Shift Problem, which can be solved in subexponential time on quantum computers. Therefore, these schemes are not practical.
  9. Paper on Attacking CSIDH-like Problems

    Constructing Elliptic Curve Isogenies in Quantum Subexponential Time

SQIsign

  1. GPSsign (2017)

    Steven D. Galbraith, Christophe Petit, and Javier Silva proposed GPSsign, a signature scheme based on quaternion algebras. However, it requires multiple rounds, resulting in relatively long signatures.

    Identiﬁcation Protocols and Signature Schemes Based on Supersingular Isogeny Problems

  2. SQIsign (2020)

    Luca De Feo, David Kohel, Antonin Leroux, Christophe Petit, and Benjamin Wesolowski proposed SQIsign, a signature scheme based on quaternion algebras. It has the shortest public key and signature sizes among NIST's post-quantum signature candidates. It primarily uses the Sigma protocol and the KLPT algorithm for finding equivalent ideals.

    SQISign: Compact Post-quantum Signatures from Quaternions and Isogenies

  3. Summary

    As a second-round candidate in NIST's post-quantum signature competition, SQIsign has the smallest public key and signature sizes. It does not leak image points and is not vulnerable to Kuperberg's quantum attack due to its quaternion algebra operations. However, its low efficiency is a drawback.

Optimizations of SQIsign

  1. Fast SQIsign (2023)

    Luca De Feo, Antonin Leroux, Patrick Longa, and Benjamin Wesolowski modified the method for computing elements in quaternion algebras to ensure uniform output in the first step of SQIsign. They also optimized the ideal-to-isogeny translation process.

    New Algorithms for the Deuring Correspondence: Towards Practical and Secure SQISign Signatures

  2. Accelerating SQIsign (2024)

    Kaizhan Lin, Weize Wang, Zheng Xu, and Changan Zhao further optimized the ideal-to-isogeny translation process in SQIsign using norm-trace relations and pairings.

    A Faster Software Implementation of SQIsign

  3. SQIsign over Extension Fields (2024)

    Maria Corte-Real Santos, Jonathan Komada Eriksen, Michael Meyer, and Krijn Reijnders extended the base field of rational points in SQIsign to increase the number of available points, thereby accelerating the protocol.

    ApresSQI: Extra Fast Verification for SQIsign Using Extension-Field Signing

Variants of SQIsign

  1. SQIsignHD (2024)

    Pierrick Dartois, Antonin Leroux, Damien Robert, and Benjamin Wesolowski removed the requirement for smooth isogeny degrees using SIDH attack techniques. Instead, they compute image points of the output isogeny and recover them during verification using a 4-dimensional SIDH attack. The KLPT algorithm is modified to randomly select equivalent ideals since smoothness is no longer required.

    SQIsignHD: New Dimensions in Cryptography

  2. SQIsign2D-West (2024)

    Andrea Basso, Pierrick Dartois, Luca De Feo, Antonin Leroux, Luciano Maino, Giacomo Pope, Damien Robert, and Benjamin Wesolowski use 2-dimensional isogenies to translate ideals to isogenies, construct auxiliary isogenies, and complete verification. Non-smooth 1-dimensional isogenies are computed using 2-dimensional techniques.

    SQIsign2D–West: The Fast, the Small, and the Safer

  3. SQIsign2D-East (2024)

    Kohei Nakagawa, Hiroshi Onuki, Wouter Castryck, Mingjie Chen, Riccardo Invernizzi, Gioella Lorenzon, and Frederik Vercauteren differ from West in that East does not translate ideals to isogenies. Instead, it uses properties of Eichler orders to directly find auxiliary isogenies on the public-key curve, still using 2-dimensional isogeny techniques.

    SQIsign2D-East: A New Signature Scheme Using 2-Dimensional Isogenies

  4. SQIsign2D\\(^2\\)(2025)

    Zheng Xu, Kaizhan Lin, Chang-An Zhao, and Yi Ouyang reduce 2-dimensional isogeny computations by leveraging 1-dimensional isogenies compared to East.

    SQIsign2D\\(^2\\): New SQIsign2D Variant by Leveraging Power Smooth Isogenies in Dimension One

  5. SQIsign2DPush (2025)

    Kohei Nakagawa and Hiroshi Onuki differ from East in that SQIsign2DPush finds an isogeny of degree d on the initial curve and pulls it back to form the auxiliary isogeny.

    SQIsign2DPush: Faster Signature Scheme Using 2-Dimensional Isogenies

  6. Summary

    Through these optimizations and variants, the efficiency of the SQIsign family has been significantly improved, making it competitive in post-quantum signatures.

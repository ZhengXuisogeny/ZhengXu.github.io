CSIDH

CSIDH(Commutative Supersignular Isogeny Diffie Hellman)

Isogeny-based cryptography is a type of cryptographic technique that relies on elliptic curve isogenies and offers the advantage of short sizes.

CSIDH

  1. Origin (2018)

    Wouter Castryck, Tanja Lange, Chloe Martindale, Lorenz Panny, and Joost Renes first proposed CSIDH (Commutative SIDH), a key exchange protocol based on supersingular elliptic curve isogenies over \\(\\mathbb{F}_p\\). CSIDH uses elliptic curves over finite fields \\(\\mathbb{F}_p\\), similar to ordinary curves with complex multiplication, and involves the action of ideal class groups (\\(\\mathbb{Z}[\\pi_p]\\) or \\(\\mathbb{Z}[\\frac{1+ \\pi_p}{2}]\\)), which are vulnerable to quantum attacks.

    CSIDH: An Efficient Post-Quantum Commutative Group Action

  2. Paper on Attacking CSIDH-like Problems

    Constructing Elliptic Curve Isogenies in Quantum Subexponential Time

  3. Signature Schemes CSI-FiSh and SeaSign (2019)

    Ward Beullens, Thorsten Kleinjung, and Frederik Vercauteren proposed CSI-FiSh, a signature scheme based on CSIDH. Luca De Feo and Steven D. Galbraith proposed SeaSign. The main difference is that CSI-FiSh uses a cyclic ideal class group.

    CSI-FiSh: Efficient Isogeny Based Signatures Through Class Group Computations

    SeaSign: Compact Isogeny Signatures from Class Group Actions

  4. SiGamal Encryption (2020)

    Ward Beullens, Thorsten Kleinjung, and Frederik Vercauteren proposed CSI-FiSh, a signature scheme based on CSIDH. Luca De Feo and Steven D. Galbraith proposed SeaSign. The main difference is that CSI-FiSh uses a cyclic ideal class group.

    SiGamal: A Supersingular Isogeny-Based PKE and Its Application to a PRF

  5. Attack on the DDH Problem in CSIDH (2020)

    Wouter Castryck, Jana Sotakova, and Frederik Vercauteren attacked the DDH problem in CSIDH using quadratic characters (genus theory).

    Breaking the Decisional Diﬃe-Hellman Problem for Class Group Actions Using Genus Theory

  6. Attacking CSIDH Using SIDH Techniques (2023)

    Wouter Castryck, Marc Houben, Simon-Philipp Merz, Marzio Mula, Sam van Buuren, and Frederik Vercauteren used correspondences between groups generated by eigenvectors of the Frobenius endomorphism, computed torsion points by pairing, to attack CSIDH using SIDH techniques. However, due to limitations of the Tate pairing, this attack is not effective.

    Weak Instances of Class Group Action Based Cryptography via Self-pairings

  7. Extending attacks via sesquilinear pairings (2024)

    Joseph Macula and Katherine E. Stange improved the above attack from sesquilinaer pairings

    Extending Class Group Action Attacks via Sesquilinear Pairings

  8. OSIDH (2020)

    Leonardo Colò and David Kohel proposed OSIDH, which differs from CSIDH in that it uses an order of an imaginary quadratic field instead of \\(\\mathbb{Z}[\\pi_p]\\) or \\(\\mathbb{Z}[\\frac{1+ \\pi_p}{2}]\\) for its endomorphism ring.

    Orienting Supersingular Isogeny Graphs

  9. Attack on the DDH Problem in OSIDH (2022)

    Wouter Castryck, Marc Houben, Frederik Vercauteren, and Benjamin Wesolowski attacked the DDH problem in OSIDH.

    On the Decisional Diffie–Hellman Problem for Class Group Actions on Oriented Elliptic Curves

  10. PEGASIS: Practical Effective Class Group Action using 4-Dimensional Isogenies (2025)

    Pierrick Dartois, Jonathan Komada Eriksen, Tako Boris Fouotsa, Arthur Herlédan Le Merdy, Riccardo Invernizzi, Damien Robert, Ryan Rueger, Frederik Vercauteren, Benjamin Wesolowski attacked the DDH problem in OSIDH.

    PEGASIS: Practical Effective Class Group Action using 4-Dimensional Isogenies

  11. The hardness problems in CSIDH and OSIDH reduce to the Hidden Shift Problem, which can be solved in subexponential time on quantum computers. Therefore, these schemes are not practical.
